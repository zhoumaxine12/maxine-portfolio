Experience detail panel SVG pack

Recommended usage
1) Quote area
- Use CSS for the main quote container.
- Use quote-mark-left.svg and quote-mark-right.svg as decorative quote marks.
- Use quote-box-corner-*.svg for the four corner accents.
- Do not stretch a whole-box SVG background.

2) Tools area
- Use these as non-official, style-consistent icons for the tools section.
- Recommended size: 40-56px icon, label below.
- Files:
  - tool-excel.svg
  - tool-sql.svg
  - tool-python.svg
  - tool-tableau.svg
  - tool-chatgpt-codex.svg

3) Naming
All assets are designed to match a clean pixel portfolio style with black / white / blue as the main system.
