Coding Works SVG asset pack

Suggested usage
1. Feature icons (right content panel)
- icon-feature-nlp.svg
- icon-feature-board.svg
- icon-feature-reminder.svg
Use these beside the three product highlight bullets.

2. Demo CTA
- icon-open-link.svg: use inside or next to the “体验 Demo” button.
- icon-demo-badge.svg: optional small accent icon if needed.

3. Preview carousel
- icon-carousel-left.svg
- icon-carousel-right.svg
Use as previous/next buttons for screenshot switching.

4. Window/header accents
- icon-window-plus.svg: use on the preview window title bar right corner.
- icon-thumbnail-active-corner.svg: optional corner accent for active thumbnail/tab.

Design notes
- Keep the page in a clean pixel-portfolio style.
- Use CSS for frames, tabs, shadows, and buttons.
- Use SVGs only for the icons/accent details; do not convert whole panels into SVG.
